# Roadmap

Eclipse is an active private research system. Public roadmap notes are intentionally high level.

## Current Focus

- Stronger separation between real low-cap snipes and low-cap bait
- Faster wallet-event ingestion for hot market events
- Better wallet funding-graph learning
- More accurate narrative-runner handling for exceptional lore or tech-driven tokens
- Cleaner public-source social context attachment
- Safer paper-trade evaluation loops

## Ongoing Principles

- Keep private runtime data out of public repos.
- Prefer compact learning summaries over raw logs.
- Treat paper-trade outcomes as feedback, not proof of future performance.
- Keep operator controls and safety gates visible.
- Use public documentation to explain the system without exposing the system.
