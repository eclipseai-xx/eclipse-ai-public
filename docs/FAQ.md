# FAQ

## Is Eclipse open source?

No. This repository is a public overview only. The private runtime and backend source code are not published.

## Does this repo contain trading logic?

No. It describes high-level system areas but does not include executable strategy code, private thresholds, wallet lists, or backend implementation details.

## Does Eclipse control real funds?

This public repository describes the system at a high level and references paper-trade simulation. It does not publish private wallet material, transaction signing logic, or operational trading infrastructure.

## Is this financial advice?

No. Eclipse is experimental software. Nothing in this repository is financial advice or trading advice.

## Why publish this repo?

To provide a public identity and technical overview for Eclipse without exposing private backend code, secrets, wallet data, prompts, or operational strategy.
