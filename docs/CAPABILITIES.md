# Capabilities

Eclipse combines autonomous agent orchestration with market intelligence, wallet analytics, memory, and operator-controlled social automation.

## Market Intelligence

- Public token launch and migration monitoring
- Post-graduation market follow-through tracking
- Market-cap, liquidity, and volume-aware candidate review
- Fast scalp and longer narrative-runner paper-trade simulation
- Outcome-based learning from simulated entries and exits

## Wallet Analytics

- Tracked-wallet event monitoring
- Wallet reputation scoring from historical behavior
- Buy/sell round-trip learning
- Co-buyer discovery around profitable moves
- Funding-graph analysis for fresh-wallet and same-funder patterns
- Probation and graduation flows for newly discovered wallets

## Narrative Intelligence

- Internet-culture and meme-lore classification
- Public social/news context tracking
- Source-aware viral-topic interpretation
- Separation of generic noise from stronger lore or tech narratives
- Memory-backed learning from prior narrative outcomes

## Risk Controls

- Holder concentration review
- Suspicious cluster and funding-pattern review
- Liquidity and volume sanity checks
- Market-cap tier awareness
- Paper-trade-only evaluation paths for strategy testing

## Social Automation

- Operator-controlled post generation
- Source-attached opinions through quote/retweet/source URL context
- Reply guards to avoid self-replies, duplicate replies, and spam patterns
- Thread-safe posting for longer thoughts

## Agent Runtime

- Local-first runtime design
- Long-running memory and coordination state
- Browser automation lanes for public-source research
- Shared-brain decision capture across agent tools
- Health/status surfaces for runtime monitoring
